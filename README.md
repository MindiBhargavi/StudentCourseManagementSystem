# Student Course Management System

## Project Overview

The Student Course Management System is a console-based Java application developed using Object-Oriented Programming (OOP) concepts. It allows users to manage student and course information and assign courses to students.

This project demonstrates the practical implementation of Java OOP principles such as Classes, Objects, Constructors, Encapsulation, Inheritance, Polymorphism, and Abstraction.

---

## Features

- Add Student Details
- Add Course Details
- Assign Course to Student
- Display Student Details
- Display Course Details
- Display All Students
- Display All Courses
- Menu-driven Console Application

## OOP Concepts Used

### Class and Object
- Person
- Student
- Course
- StudentManager
- CourseManager
- Main

### Constructor
Constructors are used to initialize Student and Course objects.

### Encapsulation
All data members are declared as private and accessed using getters and setters.

### Inheritance
The Student class extends the Person class to inherit common attributes.

### Polymorphism
The `displayDetails()` method is implemented differently in Student and Course classes.

### Abstraction
The Display interface defines the `displayDetails()` method, which is implemented by Student and Course classes.
